# Effective MQTT 前端模块

这是 Effective MQTT 项目的前端模块，使用 React 构建。

## 项目结构

- `src/components`：包含可复用的 React 组件。
- `src/pages`：包含页面级组件。
- `src/assets`：包含静态资源（如图片和样式）。
- `src/utils`：包含工具函数。
- `src/services`：包含用于 API 调用和后端交互的服务。

## 前置条件

- Node.js（建议使用 v16 或更高版本）
- npm（建议使用 v8 或更高版本）

## 快速开始

### 安装依赖

运行以下命令安装项目依赖：

```bash
npm install
```

### 启动开发服务器

运行以下命令启动开发服务器：

```bash
npm start
```

应用程序将可通过 `http://localhost:3000` 访问。

### 构建生产环境

运行以下命令构建生产环境：

```bash
npm run build
```

生产环境的文件将生成在 `build` 目录中。
